# 📋 Programa de Notas

> Aplicación web para gestionar y analizar las notas de un curso de 10 alumnos.

---

## ✨ Funcionalidades

- **Registro de alumnos** — Ingresa nombre y notas de 3 certámenes por alumno (máx. 10)
- **Validación de datos** — Verifica que las notas estén entre 1 y 100
- **Promedios automáticos** — Calcula el promedio individual, por certamen y general del curso
- **Aprobados y reprobados** — Clasifica alumnos con promedio ≥ 55 (aprobado) o < 55 (reprobado)
- **Ranking** — Ordena los alumnos de mayor a menor promedio con 🥇🥈🥉

---

## 🛠️ Tecnologías

`HTML` `CSS` `JavaScript`

---

## 🚀 Cómo usarlo

1. Abre `index.html` en tu navegador
2. Ingresa el nombre y las 3 notas de cada alumno
3. Presiona **"Agregar Alumno"** (hasta completar los 10)
4. Presiona **"Ver Resultados"** para ver el análisis completo
